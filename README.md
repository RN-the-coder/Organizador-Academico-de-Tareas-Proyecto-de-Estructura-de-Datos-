# Organizador-Academico-de-Tareas-Proyecto-de-Estructura-de-Datos-
Aplicación en Python para la gestión de tareas académicas, utilizando pandas para manipulación tabular y deque como estructura LIFO para el historial. Permite registrar, ordenar y completar tareas, con persistencia en archivos CSV y un menú interactivo en consola para control eficiente del flujo de datos.
